# CLI To-Do List 📝

A simple command-line to-do list app built with Python. Tasks are saved to a JSON file so they persist between sessions.

## Features
- Add, delete, and list tasks
- Mark tasks as done
- Data saved automatically to `tasks.json`

## Requirements
- Python 3.x

## How to run
```bash
python cli_project.py
```

## Commands
| Command | Description |
|---------|-------------|
| `add`    | Add a new task |
| `delete` | Delete a task |
| `list`   | View all tasks |
| `done`   | Mark a task as done |
| `exit`   | Exit the program |
